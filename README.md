# 📦 AI React UI Component Library — NPM Package + SaaS Platform

A production-ready **AI-powered React UI Component Library** that allows developers to install reusable UI components via NPM or generate custom UI components using AI.

This project demonstrates how to build, publish, and monetize a developer tool using modern SaaS architecture with authentication, payments, and AI integration.

---

## 🚀 Features

* Publish your own React UI Library to NPM
* AI-powered component generation
* Ready-to-use reusable UI components
* Subscription-based payment system
* Authentication and user management
* Dynamic component creation using prompts
* Full MERN Stack backend integration

---

## 🛠 Tech Stack

Frontend:

* React.js
* Tailwind CSS

Backend:

* Node.js
* Express.js
* MongoDB

AI Integration:

* OpenRouter API

Payments:

* Razorpay Integration

Package Management:

* NPM Package Publishing

